　　　　　　S2G Ver1.2

■■■■■■■■■■■■■■■■■■■■■■■■■■■■

この度は「S2G」を手に取っていただき、誠にありがとうございます。
S2G.exeを実行することでゲームが起動します。

ゲーム制作の最初の一歩として開発した作品です。
稚拙な作品ですが、赤子が初めて歩いたような気持ちで、
プレイしていただければ幸いです。

こちらは「OpenSiv3D v0.6.9 」を利用して製作されております。
アプリケーションを実行するためには、アプリ動作システム要件を満たす必要があります。
システム要件は、下記URLまたは　Siv3D公式サイト->ダウンロード->(プレイ環境)->
1.2 アプリ動作システム要件　を参照してください。
https://siv3d.github.io/ja-jp/download/windows/
■■■■■■■■■■■■■■■■■■■■■■■■■■■■
・操作方法
移動：キーボードのカーソルキー、1クリック＝1歩移動
ゲーム終了：キーボード[Esc]、ウィンドウの右上にある[×]
？：スペースキー
